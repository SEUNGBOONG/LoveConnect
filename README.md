# 💕 LoveConnect

> **재회 가능성을 열어주는 웹앱**
> 
> 잃어버린 인연을 다시 만날 수 있는 기회를 제공합니다.

---

## ✨ 프로젝트 소개

LoveConnect는 헤어진 연인이나 과거의 인연들과 다시 만날 수 있는 기회를 제공하는 혁신적인 웹 애플리케이션입니다. 우리는 모든 사람이 사랑을 다시 시작할 기회를 가질 자격이 있다고 믿습니다.

---

## 🎯 주요 기능

- 💌 **스마트 매칭 시스템** - AI 기반의 지능형 관계 매칭
- 🔐 **안전한 프라이버시 보호** - 사용자의 개인정보 완전 보호
- 💬 **실시간 채팅** - 매칭된 사람과의 부드러운 대화
- 🎭 **관계 평가 시스템** - 과거 관계의 호환도 분석
- 🌟 **프로필 커스터마이징** - 자신을 최고로 표현하세요
- 🎁 **특별한 순간 기념** - 재회의 기념일을 축하해보세요

## 📦 설치 및 실행

### 필수 요구사항
- Node.js (v14 이상)
- npm 또는 yarn

### 설치 방법

```bash
# 저장소 클론
git clone https://github.com/SEUNGBOONG/LoveConnect.git

# 디렉토리 이동
cd LoveConnect

# 의존성 설치
npm install

# 개발 서버 시작
npm start
```
